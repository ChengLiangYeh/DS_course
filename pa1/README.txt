test.py是邊寫main檔時的測試檔，寫成功test檔才將他改寫入main檔中
使用方式直接: python test.py

main檔:繳交作業需要給的檔案，最終完成的檔案。有使用到argparse，#####中間夾著
的是自己寫的程式碼:如何讀input，再把input輸入到struct中使用stack or queue。
使用main檔的方式:argparse
python main.py --structure stack --input ./input.txt --output ./output.txt
python main.py --structure queue --input ./input.txt --output ./output.txt
input可以自己改
